# Student Registration Web Application

A simple Java-based web application using **Servlets**, **JSP**, and **MySQL** to register students and view all registered students.

---

## Features

- Register a new student with **name**, **email**, and **year**.
- Validate user input.
- Prevent duplicate email registrations.
- View all registered students.
- Simple and clean JSP interface.

---

## Technologies

- **Java Servlet API (Jakarta Servlet)**
- **JSP**
- **MySQL**
- **JDBC**
- **HTML/CSS**
